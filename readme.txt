Seamlessly integrate your WooCommerce store with popular marketplaces like Trendyol, N11, Hepsiburada, Amazon, Etsy, Shopify, and Pazarama.

== Description ==

API ISARUD is a powerful WooCommerce plugin that simplifies management by integrating your online store with popular marketplaces. With support for Trendyol, N11, Hepsiburada, Amazon, Etsy, Shopify, and Pazarama, you can directly manage inventory, synchronize prices, track orders, and match products from your WordPress admin panel.  

**Key Features:**
* Centralized inventory management across all marketplaces
* Automatic inventory synchronization
* Bulk product updates with variable product support  
* Order tracking and status updates
* Secure API connections with OAuth 2.0 and REST/SOAP support
* User-friendly admin panel with detailed statistics and error logs  
* Multi-language support (Turkish and English)

== External Services ==

This plugin connects to various marketplace APIs to synchronize product data:

= Trendyol =
* Service: Trendyol Seller Center API
* Purpose: Product and inventory management
* Data Sent: Product details, stock levels, prices
* API Documentation: https://developers.trendyol.com/  
* Terms of Service: https://www.trendyol.com/sozlesmeler/satici-uye-sozlesmesi

= N11 =  
* Service: N11 Seller API
* Purpose: Product and inventory management
* Data Sent: Product details, stock levels, prices
* API Documentation: https://api.n11.com/ws/
* Terms of Service: https://www.n11.com/genel/satici-uyelik-sozlesmesi-76

= HepsiBurada =
* Service: HepsiBurada Marketplace API  
* Purpose: Product and inventory management
* Data Sent: Product details, stock levels, prices
* API Documentation: https://developers.hepsiburada.com/

= Etsy =

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the "API ISARUD" menu in your WordPress admin panel to configure marketplace settings
4. Match your WooCommerce products with desired marketplaces  
5. Start inventory synchronization
